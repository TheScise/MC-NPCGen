# Minecraft NPC Character Generator - Build Request

I want you to build a lightweight web application for a Minecraft livestream series.

The purpose of this app is to create entertaining NPC characters live on stream.

The streamer will use the app to generate a character with chat participation, then display the finished character sheet as an OBS Browser Source overlay.

The priority is:
1. Fast live interaction
2. Fun character creation experience
3. Simple deployment
4. Easy future expansion

Do not overengineer this.

---

# Tech Requirements

Build using:

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- No database
- No authentication
- No backend
- Static deployment compatible with Vercel

Data should live in JSON files inside the repository.

The project should work with:

npm install

npm run dev

and deploy directly to Vercel.

---

# Application Structure

The app has two main pages.

## 1. Control Panel

Route:

/

This is the streamer-facing interface.

This page is not shown to viewers.

It should allow the streamer to generate a Minecraft NPC character live.

---

## 2. OBS Overlay

Route:

/overlay

This page is designed to be added as an OBS Browser Source.

It should:

- Have no controls
- Only display the character information
- Update automatically
- Be visually clean on stream

---

# Character Generation Experience

The main interaction should feel like rolling an RPG character.

Do NOT make this a traditional form.

The user should feel like they are creating a character in a game.

Each attribute should be represented as a large animated card.

Example:

----------------------------

NAME

HAROLD

🎲 Roll

🔒 Lock

----------------------------

RACE

DWARF

🎲 Roll

🔒 Lock

----------------------------

Each card should:

- Display the current value prominently
- Have a Roll button
- Have a Lock toggle
- Animate when rolling

---

# Slot Machine Animation

When rolling:

- Animate through multiple possible values
- Stop on the final result
- Animation should last around 1 second
- Feel satisfying but not annoying

Example:

Rolling...

Human

Elf

Orc

Dwarf

Dwarf ✓

Use Framer Motion.
